## 📄 核心文件说明

### 后端文件

#### main.py
- FastAPI 应用主入口
- 定义所有 API 路由
- CORS 配置
- 错误处理

#### config.py
- 应用配置
- 环境变量加载
- 常量定义

#### routers/chat.py
- 聊天 API 路由
- 流式响应处理
- 会话管理

#### services/llm_service.py
- OpenAI API 调用
- 流式响应生成
- 错误处理

#### services/rag_service.py
- 知识库检索
- 向量数据库查询
- 文档处理

#### services/session_manager.py
- 会话管理
- 对话历史
- 会话持久化

#### utils/admin_config.py
- 管理配置管理
- System Prompt 配置
- 参数管理

#### utils/admin_persist.py
- 数据持久化
- JSON 存储
- 数据备份

#### scripts/extract_documents.py
- 文档提取
- 文本处理
- 分块处理

#### scripts/build_vectordb.py
- 向量数据库构建
- 嵌入生成
- 索引创建

## 📊 API 端点

### 聊天 API
- `POST /api/chat/stream` - 流式聊天
- `GET /api/chat/history` - 获取历史
- `DELETE /api/chat/session` - 删除会话

### 管理 API
- `GET /api/admin/config` - 获取配置
- `POST /api/admin/config` - 更新配置
- `GET /api/admin/documents` - 获取文档列表
- `POST /api/admin/documents` - 上传文档
- `DELETE /api/admin/documents` - 删除文档

## 🛠️ 开发指南

### 添加新的 API 端点

1. 在 `routers/` 中创建新文件
2. 定义路由函数
3. 在 `main.py` 中注册路由

### 添加新的服务

1. 在 `services/` 中创建新文件
2. 定义服务类
3. 在需要的地方导入使用

### 修改配置

1. 编辑 `config.py`
2. 或在 `.env` 中设置环境变量
