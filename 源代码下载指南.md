## 🚀 快速开始

### 启动后端
```bash
cd chatbot/backend
pip install -r requirements.txt
python main.py
```

### 启动管理后台
```bash
cd chatbot/admin/admin-backend
npm install
npm start
```

### 启动前端
```bash
cd chatbot/admin/admin-frontend
npm install
npm run dev
```
